# Resoluções do Beecrowd em Java e Python
# Solution to Beecrowd problems in Java and Python

# PORTUGUÊS_BRASILEIRO
 Um método de aprendizado utilizando a resolução de atividades em duas linguagens de grande interesse (Java e Python), realizando gradualmente todas as atividades.
 Para rodar é simples: crie um arquivo com o nome da classe se for em Java, e copie o código. *Se precisar utilizar no Beecrowd, renomeie a classe para Main, pois o Beecrowd apenas aceita este nome para a classe!*
 
# ENGLISH
 A learning strategy using problem solving in two languages of great interest (Java and Python), gradually carrying out all activities.
 It's easy to use: make a file with the class name if it is in Java and copy the code. *If you need to submit it on Beecrowd, rename the class to Main, because Beecrowd only accepts classes named "Main"!*
